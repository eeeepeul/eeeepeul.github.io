# Project Identity

This project is not a traditional developer blog.

It is an experimental visual magazine inspired by:
- kiiikiii.kr
- fashion campaign websites
- editorial layouts
- interactive archives
- cinematic digital experiences

The blog system exists only as a content structure.
The primary goal is visual storytelling and atmosphere.

# Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MDX
- Framer Motion allowed

# Design Direction

Prefer:
- oversized typography
- image-heavy layouts
- luxurious spacing
- cinematic transitions
- experimental compositions
- editorial magazine aesthetics
- immersive landing pages
- visual hierarchy
- minimal but emotional UI

Avoid:
- generic SaaS UI
- dashboard layouts
- developer blog aesthetics
- dense layouts
- overuse of cards
- overly technical appearance
- template-looking sections

# Layout Principles

The website should feel like:
- a curated visual archive
- a fashion editorial
- an interactive campaign
- a premium digital publication

Each page should have:
- strong visual rhythm
- intentional whitespace
- smooth transitions
- responsive layouts
- elegant typography

# Motion

Motion should feel:
- subtle
- smooth
- cinematic
- premium

Avoid excessive animations.

Use motion mainly for:
- hover interactions
- fade transitions
- staggered entrances
- image movement
- typography reveals

# Development Rules

Preserve:
- MDX post structure
- routing system
- SEO functionality
- responsive behavior

Prefer:
- reusable components
- simple architecture
- minimal dependencies
- clean Tailwind usage

# Priority

1. Visual identity
2. Typography
3. Layout composition
4. Motion
5. Blog functionality
6. Performance
